# Color Palette Generator

A modern color palette generator that delivers visually balanced, designer-friendly color combinations with one-click copying, responsive UI, and a curated palette database of 400+ combinations.

---

## Overview

Choosing colors sounds easy тАФ until you actually start designing.

Whether you're building:

- Websites
- UI components
- Posters
- Cards
- Branding assets
- Dashboards

finding colors that truly work together is often harder than expected.

Most random color generators simply output unrelated colors with no visual harmony, forcing developers and designers to manually test combinations until something finally looks right.

Color Palette Generator was built to solve that problem.

Instead of generating completely random colors, this project provides carefully selected color palettes where every color is intentionally grouped to blend naturally with the others.

Each palette is designed to feel:

- Balanced
- Smooth
- Visually cohesive
- Easy on the eyes
- Practical for real-world design usage

The project combines simplicity, usability, and interaction-focused design into a lightweight tool that helps users discover usable color systems instantly.

## Usage

Using the application is intentionally simple.

### Generate a Palette

- Open the application
- A default palette is displayed immediately
- Click the Generate Palette button
- Instantly receive a new curated 5-color palette

Repeat the process as many times as needed until you discover a palette that fits your project.

---

### Copy Colors

- Hover over any color card
- Click the card
- The HEX color code is automatically copied to your clipboard
- A visual confirmation message appears after copying

This allows fast palette extraction without interrupting workflow.

## How It Works

The project uses a curated JSON database containing hundreds of predefined color palettes.

When users generate a palette:

- A palette is selected randomly from the database
- All five colors are rendered dynamically
- Each color card becomes individually copyable
- UI feedback updates instantly after interaction

This ensures that generated palettes maintain intentional visual consistency instead of relying on purely random color generation.
